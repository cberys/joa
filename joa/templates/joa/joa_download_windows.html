Java OceanAtlas Application

Windows Installation Instructions

PLEASE NOTE: In order to run JOA 5 on Windows, you must have a working Java VM installed.

Start at: http://www.java.com/en/

Click the Free Java Download button to download a JVM to your PC. The download page will determine the appropriate version of Java to download. Once downloaded, follow all installation instructions to install Java on your PC.

This link will tell you what version of Java you have installed (if any): http://www.java.com/en/download/installed.jsp

Installation

After downloading, just double-click installer icon to begin installation process:

Note: The installation file you download will have a _32 or _64 at the end of the file name depending on what version of the installer you have downloaded.

You will be presented with a series of dialogs to guide you through the installation process:

Select an installation folder if different than the default:

Click Install to install JOA onto your PC:

Click Finish at the end of the installation process:

Windows System Requirements

Java 6.0 (1.6)
Microsoft Windows XP, Vista, and 7
4 GB of RAM
