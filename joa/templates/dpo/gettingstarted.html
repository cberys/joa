Descriptive Physical Oceanography Examples

Get Started

To get JOA:

  • Go to the Install JOA page for instructions, links, and tips on getting and using Java OceanAtlas.

  • You can also go directly to http://joa.ucsd.edu if you are already familiar with the application. IMPORTANT: Remember to COPY the Java OceanAtlas Application and support files to your hard drive to complete the installation process.

To use the data files:

  • Data files can be opened from within JOA
  • Go to the Data Files page to copy files to your work area.
  • With JOA open, go to the File menu, and select Open. Select the file

To navigate the DPO JOA Examples:

  • Go to the DPO JOA Examples chapter list.
  • Select the chapter you would like to work on.
  • Use the navigation bar at left to navigate within or between chapters.
